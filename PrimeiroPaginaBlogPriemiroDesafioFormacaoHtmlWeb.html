Ferramentas utilizadas

Instalação e Preparação do VS Code para HTML

Passo 1: Baixar o VS Code

  1. Abra o seu navegador de internet (Chrome, Firefox, Edge, etc.).
  2. Acesse o site oficial do Visual Studio Code: https://code.visualstudio.com/
  3. O site deve detectar automaticamente o seu sistema operacional e exibir o botão de download correto.
  4. Clique no botão para baixar a versão estável.
  5. Aguarde o download do arquivo de instalação ser concluído.

Passo 2: Instalar o VS Code

Windows:
  • Localize o arquivo .exe baixado (geralmente em "Downloads").
  • Dê um duplo clique no arquivo para iniciar a instalação.
  • Leia e aceite o contrato de licença.
  • Clique em "Avançar" nas telas seguintes, mantendo as opções padrão.
  • Marque a opção "Adicionar ao PATH".
  • Clique em "Instalar" e aguarde a conclusão.
  • Clique em "Concluir" para abrir o VS Code.
macOS:
  • Localize o arquivo .dmg baixado (geralmente em "Downloads").
  • Dê um duplo clique no arquivo .dmg.
  • Arraste o ícone do VS Code para a pasta "Applications".
  • Abra a pasta "Applications" e clique duas vezes no ícone do Visual Studio Code para iniciá-lo.
Linux:
  • Escolha o arquivo adequado para sua distribuição (.deb, .rpm, .tar.gz).
  • Para .deb (exemplo Ubuntu): Abra o terminal, navegue até a pasta e execute: sudo dpkg -i <nome_do_arquivo>.deb, seguido de sudo apt-get update && sudo apt-get install -f.
  • Para .rpm (exemplo Fedora): Abra o terminal, navegue até a pasta e execute: sudo rpm -i <nome_do_arquivo>.rpm.
  • Para .tar.gz: Descompacte e execute o arquivo code dentro da pasta. Considere criar um link simbólico.

Passo 3: Preparar o VS Code para HTML

  1. Abrir uma pasta de projeto (recomendado):
    • Crie uma pasta no seu computador (ex: "meu_projeto_web").
    • Abra o VS Code.
    • Vá em "Arquivo" (File) > "Abrir Pasta..." (Open Folder...).
    • Selecione a pasta e clique em "Selecionar Pasta" (Select Folder).
  2. Criar um arquivo HTML:
    • Na barra lateral esquerda, clique no ícone de "Novo Arquivo".
    • Digite o nome do arquivo com a extensão .html (ex: index.html) e pressione Enter.
  3. Escrever seu código HTML:
    • Comece a digitar a estrutura HTML básica (experimente digitar ! e pressionar Tab). 
      • 
<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Minha Página HTML</title>
</head>
<body>
    <h1>Olá, Mundo!</h1>
    <p>Este é meu primeiro código HTML no VS Code.</p>
</body>
</html>
  4. Salvar o arquivo:
    • Vá em "Arquivo" (File) > "Salvar" (Save) ou pressione Ctrl + S (Windows/Linux) / Cmd + S (macOS).
  5. Visualizar no navegador:
    • Abra a pasta onde você salvou o arquivo .html.
    • Dê um duplo clique no arquivo para abri-lo no navegador.

Dicas Extras para HTML no VS Code:

  • Emmet: Já integrado, facilita a escrita de HTML e CSS com abreviações (ex: digite li*3 e pressione Tab).
  • Extensões (opcional, mas útil):
    • HTML CSS Support: Autocompletar para classes e IDs CSS.
    • Auto Rename Tag: Renomeia tags de abertura e fechamento simultaneamente.

Usando o Inspetor de Elementos

(Voltar)

O Inspetor de Elementos é uma ferramenta essencial, integrada diretamente nos navegadores web modernos como Google Chrome, Mozilla Firefox, Microsoft Edge e Apple Safari. Ele serve como uma janela para o "código de bastidores" de qualquer página web que você visita. Através dele, é possível examinar e até mesmo modificar temporariamente a estrutura HTML (que define o conteúdo e a organização dos elementos na página) e o CSS (que controla a apresentação visual desses elementos, como cores, fontes e layout).

Para que serve o Inspetor de Elementos?

  • Entender a estrutura HTML: Ele permite visualizar a hierarquia de elementos HTML, mostrando como as diferentes tags estão aninhadas e organizadas para formar a página. Você pode expandir e recolher nós para explorar a estrutura em detalhes.
  • Analisar o CSS aplicado: Para cada elemento HTML selecionado, o Inspetor de Elementos exibe as regras de CSS que estão afetando sua aparência. Isso inclui estilos definidos em arquivos CSS externos, dentro da tag <style> no próprio HTML e estilos inline (no atributo style).
  • Modificação em tempo real: Uma das funcionalidades mais poderosas é a capacidade de editar o HTML e o CSS diretamente na interface do Inspetor. As alterações são refletidas instantaneamente na visualização da página no seu navegador. É importante lembrar que essas modificações são apenas locais e temporárias; ao recarregar a página, o conteúdo original é restaurado.
  • Depuração de Layout e Estilo: Se um elemento não está aparecendo ou se comportando como esperado visualmente, o Inspetor de Elementos ajuda a identificar quais regras de CSS estão causando o problema. Você pode desativar ou modificar estilos para testar diferentes soluções.
  • Aprendizado e Inspeção de Outros Sites: Ao navegar por diferentes sites, você pode usar o Inspetor de Elementos para aprender como eles são construídos e estilizados. Isso pode ser uma ótima fonte de inspiração e conhecimento.
  • Análise de Desempenho (em algumas abas): Ferramentas relacionadas no mesmo conjunto (como a aba "Rede" ou "Performance") permitem analisar o tempo de carregamento de recursos e outros aspectos do desempenho da página.

Como acessar o Inspetor de Elementos?

Existem várias maneiras de abrir o Inspetor de Elementos:

  1. Clique com o botão direito do mouse: A maneira mais comum é clicar com o botão direito sobre o elemento específico que você deseja examinar (ou em qualquer lugar da página) e selecionar a opção "Inspecionar" ou "Inspecionar Elemento" no menu de contexto que aparece.
  2. Atalhos de teclado: A maioria dos navegadores oferece atalhos de teclado para abrir o Inspetor de Elementos:
    • Google Chrome, Microsoft Edge, Brave: Pressione a tecla F12 ou a combinação de teclas Ctrl + Shift + I (no Windows e Linux) ou Cmd + Option + I (no macOS).
    • Mozilla Firefox: Pressione a tecla F12 ou a combinação de teclas Ctrl + Shift + I (no Windows e Linux) ou Cmd + Option + I (no macOS).
    • Apple Safari: Primeiro, você precisa ativar o menu "Desenvolvedor" nas preferências do Safari. Vá em "Safari" > "Preferências..." > "Avançado" e marque a caixa "Mostrar menu Desenvolvedor na barra de menus". Depois disso, você pode clicar com o botão direito e selecionar "Inspecionar Elemento" ou usar o atalho Option + Cmd + I.

O que você vê ao abrir o Inspetor de Elementos?

Ao abrir o Inspetor, geralmente você verá a janela do navegador dividida em duas seções principais:

  • Aba "Elementos" (ou "Elements"): Esta aba mostra a estrutura HTML da página como uma árvore hierárquica de elementos. Você pode navegar por essa árvore, expandindo e recolhendo os nós para ver a relação entre as tags. Ao selecionar um elemento nesta aba, ele é destacado visualmente na página web.
  • Aba "Estilos" (ou "Styles"): Localizada geralmente ao lado ou abaixo da aba "Elementos", esta seção exibe as regras de CSS que estão sendo aplicadas ao elemento atualmente selecionado na aba "Elementos". Você pode ver as propriedades CSS, seus valores, e em qual arquivo ou tag de estilo cada regra está definida.

Em resumo:

O Inspetor de Elementos é uma ferramenta indispensável para qualquer pessoa que trabalha com desenvolvimento web. Ele oferece uma visão profunda de como as páginas são construídas e estilizadas, permitindo inspeção, análise e experimentação em tempo real.


Estrutura básica do HTML

(Voltar)

O que é a estrutura básica do HTML?

Todo documento HTML válido segue uma estrutura fundamental. Essa estrutura garante que o navegador entenda como interpretar e exibir o conteúdo da página corretamente. Os elementos essenciais são:

  • <!DOCTYPE html>: A declaração DOCTYPE informa ao navegador que este é um documento HTML5. Deve ser a primeira linha do seu código HTML.
  • <html>: A tag <html> é o elemento raiz de toda página HTML. Todos os outros elementos (exceto a declaração DOCTYPE) devem estar dentro desta tag. O atributo lang dentro da tag <html> especifica o idioma principal do documento (por exemplo, lang="pt-br" para português do Brasil).
  • <head>: A tag <head> contém metadados sobre o documento HTML, que não são exibidos diretamente na página. Informações como o título da página (que aparece na aba do navegador), a codificação de caracteres, informações para SEO e links para arquivos CSS são colocados dentro da tag <head>.
  • <body>: A tag <body> contém todo o conteúdo visível da página HTML, como textos, imagens, links, vídeos e outros elementos interativos.

Exemplo de estrutura básica:


<!DOCTYPE html>
<html lang="pt-br">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content="width=device-width, initial-scale=1.0">
    <title>Título da Página</title>
</head>
<body>
    <!-- Aqui vai o conteúdo visível da sua página -->
    <h1>Meu Primeiro Título</h1>
    <p>Este é um parágrafo.</p>
</body>
</html>

Falando sobre Tags

(Voltar)

As tags HTML são os blocos de construção fundamentais de uma página web. Elas são usadas para marcar e estruturar o conteúdo, informando ao navegador como exibir cada elemento.

Principais Conceitos:

  • Estrutura: A maioria das tags HTML vêm em pares: uma tag de abertura (ex: <p>) e uma tag de fechamento (ex: </p>). O conteúdo fica entre elas.
  • Função: Cada tag tem um significado semântico e define o tipo de conteúdo que ela envolve (texto, imagem, link, lista, etc.).
  • Atributos: As tags podem ter atributos que fornecem informações adicionais sobre o elemento (ex: <a href="url">, onde href é um atributo).
  • Estrutura Básica: Todo documento HTML começa com <!DOCTYPE html>, seguido pela tag <html> que envolve as seções <head> (metadados) e <body> (conteúdo visível).
  • Tags Essenciais: Algumas tags são cruciais para a estrutura e conteúdo, como <h1> a <h6> (cabeçalhos), <p> (parágrafos), <a> (links), <img> (imagens), <ul> e <ol> (listas), e <li> (itens de lista).
  • Organização e Semântica: Usar as tags corretas ajuda a organizar o conteúdo de forma lógica e fornece significado semântico para os navegadores e mecanismos de busca. Tags como <article>, <nav>, <aside> e <footer> ajudam a definir a estrutura do conteúdo.
  • Estilização: Embora a estrutura seja definida pelo HTML, a apresentação visual é controlada principalmente pelo CSS (Cascading Style Sheets). As tags HTML servem como "ganchos" para aplicar estilos.

Em resumo:

As tags HTML são a linguagem fundamental para criar páginas web. Elas estruturam o conteúdo, definem seu significado e permitem a criação de documentos interativos e bem organizados. Entender as principais tags e seus usos é o primeiro passo essencial no desenvolvimento web.


Principais Tags HTML e o que fazem:

  • <!DOCTYPE html>

    O que faz: Declara o tipo de documento como HTML5. Deve ser a primeira linha de todo documento HTML.

  • <html>

    O que faz: Define a raiz do documento HTML. Todos os outros elementos HTML devem estar dentro desta tag.

    Atributos comuns: lang (especifica o idioma do documento).

  • <head>

    O que faz: Contém metadados sobre o documento HTML, como o título da página, links para folhas de estilo CSS, etc. As informações dentro de <head> não são exibidas diretamente na página.

    Tags comuns dentro de <head>: <meta>, <title>, <link>, <style>.

  • <meta>

    O que faz: Fornece metadados sobre o documento HTML. Pode especificar o conjunto de caracteres, a viewport, a descrição, palavras-chave, o autor e outras informações.

    Atributos comuns: charset, name, content.

  • <title>

    O que faz: Define o título do documento que aparece na aba do navegador ou na barra de título da janela.

  • <body>

    O que faz: Contém todo o conteúdo visível da página HTML (textos, imagens, links, etc.).

  • <h1> a <h6>

    O que fazem: Definem os cabeçalhos HTML. <h1> representa o cabeçalho mais importante e <h6> o menos importante.

  • <p>

    O que faz: Define um parágrafo.

  • <a>

    O que faz: Define um hiperlink (âncora) para outras páginas ou locais dentro da mesma página.

    Atributos comuns: href (especifica o destino do link), target (especifica onde abrir o link), rel.

  • <img>

    O que faz: Incorpora uma imagem na página.

    Atributos obrigatórios: src (especifica o caminho para a imagem), alt (fornece um texto alternativo para a imagem).

  • <ul>

    O que faz: Define uma lista não ordenada (com marcadores).

    Tags filhas comuns: <li> (define um item da lista).

  • <ol>

    O que faz: Define uma lista ordenada (com números ou letras).

    Tags filhas comuns: <li> (define um item da lista).

  • <li>

    O que faz: Define um item de lista dentro de listas <ul> ou <ol>.

  • <div>

    O que faz: Define uma seção ou um container para elementos HTML. É usado para estruturar o layout da página e aplicar estilos CSS.

  • <span>

    O que faz: É um container inline usado para marcar partes de texto ou outros elementos inline para aplicar estilos CSS ou manipular com JavaScript.

  • <strong>

    O que faz: Define texto com forte importância, geralmente exibido em negrito.

  • <em>

    O que faz: Define texto enfatizado, geralmente exibido em itálico.

  • <br>

    O que faz: Insere uma quebra de linha simples.

  • <hr>

    O que faz: Define uma linha horizontal para separar conteúdo.

  • <small>

    O que faz: Define texto menor.

  • <pre>

    O que faz: Define texto pré-formatado. O texto dentro desta tag é exibido com uma fonte de largura fixa e preserva espaços e quebras de linha.

  • <code>

    O que faz: Define um trecho de código de computador.

  • <aside>

    O que faz: Define conteúdo relacionado ao conteúdo principal, mas que pode ser considerado separado (como barras laterais, citações, etc.).

  • <!DOCTYPE html>

    Utilização: Usada no início do seu documento HTML para declarar a versão do HTML.

  • <html>

    Utilização: Envolve todo o conteúdo da sua página HTML.

    Atributo utilizado: lang="pt-br" para definir o idioma português do Brasil.

  • <head>

    Utilização: Contém informações como o conjunto de caracteres e a viewport.

  • <meta charset="UTF-8">

    Utilização: Define a codificação de caracteres para UTF-8.

  • <meta name="viewport" content="width=device-width, initial-scale=1.0">

    Utilização: Configura a viewport para responsividade.

  • <title>

    Utilização: Define o título da sua página.

  • <body>

    Utilização: Contém todo o conteúdo visível da sua página.

  • <h1>, <h2>, <h3>, <h4>, <h5>

    Utilização: Usadas para criar os títulos e subtítulos do seu blog.

  • <hr>

    Utilização: Usada para criar uma linha horizontal separando seções.

  • <em>

    Utilização: Usada para enfatizar texto (geralmente itálico) no seu índice.

  • <ul>

    Utilização: Usada para criar a lista não ordenada do seu índice.

  • <li>

    Utilização: Usada para definir cada item da lista no seu índice.

  • <a>

    Utilização: Usada para criar os links de navegação no seu índice e os links externos.

    Atributos utilizados: href (para o destino do link), target="_blank" (para abrir em nova aba), rel="noopener noreferrer" (para segurança).

  • <br>

    Utilização: Usada para inserir quebras de linha.

  • <small>

    Utilização: Usada para exibir o texto "Voltar" em tamanho menor.

  • <p>

    Utilização: Usada para criar os parágrafos de texto do seu blog.

  • <strong>

    Utilização: Usada para destacar texto importante em negrito.

  • <code>

    Utilização: Usada para exibir trechos de código.

  • <pre>

    Utilização: Usada para exibir texto pré-formatado, preservando espaços e quebras de linha (útil para código).

  • <span>

    Utilização: Usada como um container inline para aplicar estilos (como a cor no seu "@").

  • <ol>

    Utilização: Usada para criar listas ordenadas (como os passos de instalação do VS Code).


Atributos básicos

(Voltar)

Os atributos HTML são palavras-chave especiais usadas dentro da tag de abertura de um elemento HTML para fornecer informações adicionais sobre ele. Eles modificam o comportamento ou a aparência padrão dos elementos.

Anatomia de um Atributo:

Um atributo consiste em duas partes:

  • Nome do Atributo: Uma palavra que identifica a informação adicional que você deseja fornecer (ex: href, src, class, id).
  • Valor do Atributo: O valor que você atribui ao atributo, geralmente entre aspas duplas (") ou simples (') (ex: "https://www.exemplo.com", "minha-classe").

A sintaxe básica é: <nomeDaTag nomeDoAtributo="valorDoAtributo">Conteúdo</nomeDaTag>

Principais Atributos Básicos e seus Usos:

  • id

    O que faz: Define um identificador único para um elemento HTML dentro de um documento. O valor do atributo id deve ser único em toda a página.

    Uso: Usado para estilizar um elemento específico com CSS ou para manipulá-lo com JavaScript.

    Exemplo: <div id="principal">...</div>

  • class

    O que faz: Especifica uma ou mais nomes de classe para um elemento HTML. As classes são usadas para aplicar os mesmos estilos CSS a múltiplos elementos ou para selecionar grupos de elementos com JavaScript.

    Uso: Principalmente para estilização com CSS.

    Exemplo: <p class="texto-importante texto-azul">...</p> (múltiplas classes separadas por espaços)

  • style

    O que faz: Permite adicionar estilos CSS inline diretamente a um elemento HTML.

    Uso: Geralmente usado para aplicar estilos específicos e únicos a um elemento. Para estilos mais complexos e reutilizáveis, é preferível usar CSS em folhas de estilo separadas ou dentro da tag <style>.

    Exemplo: <p style="color: green; font-size: 16px;">...</p>

  • href

    O que faz: Especifica o destino (URL) de um hiperlink.

    Uso: Usado na tag <a> para criar links para outras páginas ou recursos.

    Exemplo: <a href="https://www.exemplo.com">Clique aqui</a>

  • src

    O que faz: Especifica o caminho (URL) para um recurso externo, como uma imagem ou um script.

    Uso: Usado nas tags <img> e <script>.

    Exemplo (imagem): <img src="imagem.jpg" alt="Descrição da imagem">

    Exemplo (script): <script src="script.js"></script>

  • alt

    O que faz: Fornece um texto alternativo para uma imagem. Este texto é exibido se a imagem não puder ser carregada e é importante para acessibilidade (leitores de tela).

    Uso: Usado na tag <img>.

    Exemplo: <img src="imagem.jpg" alt="Logotipo da empresa">

  • title

    O que faz: Define um texto que é exibido como uma dica de ferramenta (tooltip) quando o mouse passa sobre o elemento.

    Uso: Pode ser usado em várias tags para fornecer informações adicionais.

    Exemplo (link): <a href="#" title="Link para a página inicial">Home</a>

  • lang

    O que faz: Especifica o idioma do conteúdo do elemento.

    Uso: Usado na tag <html> para definir o idioma principal da página e em outras tags para indicar o idioma de partes específicas do conteúdo.

    Exemplo (idioma da página): <html lang="pt-br">

    Exemplo (idioma de uma citação): <blockquote lang="en">...</blockquote>

  • target

    O que faz: Especifica onde abrir o link.

    Uso: Usado na tag <a>.

    Valores comuns: _blank (abre em uma nova janela ou aba), _self (abre na mesma janela/aba - padrão), _parent, _top (relacionados a frames).

    Exemplo: <a href="https://www.exemplo.com" target="_blank">Abrir em nova aba</a>

  • rel

    O que faz: Especifica a relação entre o documento atual e o documento vinculado.

    Uso: Mais comumente usado na tag <a> e <link>.

    Valores comuns: noopener, noreferrer, nofollow, stylesheet.

    Exemplo: <a href="https://www.exemplo.com" target="_blank" rel="noopener noreferrer">Link seguro</a>

Considerações Importantes:

  • Os nomes dos atributos geralmente são escritos em minúsculas.
  • Os valores dos atributos devem ser colocados entre aspas (duplas ou simples). É uma boa prática ser consistente.
  • Alguns atributos são específicos para certas tags (ex: href só faz sentido na tag <a>).
  • Usar atributos semanticamente corretos e para seus devidos propósitos é importante para a acessibilidade e SEO.

Entender e usar corretamente os atributos básicos é fundamental para construir páginas HTML bem estruturadas e funcionais.


Executando uma página HTML

(Voltar)

Executar uma página HTML é o processo de visualizar o código HTML que você escreveu em um navegador web. Os navegadores são projetados para interpretar o código HTML, CSS e JavaScript e renderizar a página visualmente para o usuário.

Passos para Executar uma Página HTML:

  1. Salvar o Arquivo HTML:

    Primeiro, você precisa salvar o seu código HTML em um arquivo com a extensão .html. Por exemplo, você pode nomear o arquivo como index.html, pagina.html ou qualquer outro nome que desejar, contanto que termine com .html.

    Exemplo: Se você estiver usando um editor de texto como o VS Code, após escrever seu código, vá em "Arquivo" (File) > "Salvar" (Save) e escolha um nome para o arquivo com a extensão .html.

  2. Abrir o Arquivo HTML no Navegador:

    Existem algumas maneiras de fazer isso:

    • Usando o Explorador de Arquivos (Windows) ou Finder (macOS):
      1. Localize o arquivo .html que você salvou no seu computador.
      2. Dê um duplo clique no arquivo.
      3. O seu navegador padrão será aberto automaticamente e exibirá o conteúdo da sua página HTML.
    • Arrastando e Soltando:
      1. Abra o seu navegador web.
      2. Localize o arquivo .html no seu computador.
      3. Clique no arquivo e, mantendo o botão do mouse pressionado, arráste-o para a janela do navegador e solte-o.
      4. O navegador carregará e exibirá a página HTML.
    • Usando a opção "Abrir Arquivo" do Navegador:
      1. Abra o seu navegador web.
      2. Procure pela opção de "Arquivo" (File) no menu do navegador (geralmente no canto superior esquerdo).
      3. Clique em "Abrir Arquivo" (Open File) ou um termo similar.
      4. Uma janela de diálogo será aberta para você procurar o arquivo .html no seu computador.
      5. Selecione o arquivo e clique em "Abrir".
      6. O navegador carregará e exibirá a página HTML.

  3. Visualizar o Resultado:

    Após abrir o arquivo .html no navegador, você verá a página web sendo renderizada de acordo com as tags HTML que você escreveu. Textos aparecerão, imagens serão exibidas, links estarão clicáveis e a estrutura da página será apresentada.

  4. Inspecionar o Código (Opcional, mas Recomendado para Desenvolvimento):

    Enquanto a página está aberta no navegador, você pode usar o Inspetor de Elementos (como explicado anteriormente) para ver o código HTML que o navegador interpretou e os estilos CSS que foram aplicados. Isso é fundamental para depurar e entender como a página está sendo renderizada.

    Para abrir o Inspetor de Elementos, clique com o botão direito em qualquer lugar da página e selecione "Inspecionar" ou "Inspecionar Elemento" (o texto exato pode variar dependendo do navegador), ou use os atalhos de teclado (F12, Ctrl+Shift+I, Cmd+Option+I).

Em Resumo:

Executar uma página HTML é simplesmente abrir o arquivo .html com um navegador web. O navegador fará o trabalho de ler e exibir o conteúdo da sua página para o usuário.


Textos

(Voltar)

O HTML oferece diversas tags para estruturar e formatar textos em páginas web. A escolha da tag correta é importante tanto para a apresentação visual quanto para a semântica do documento, facilitando a interpretação por navegadores, mecanismos de busca e leitores de tela.

Principais Tags para Textos:

  • <h1> a <h6> (Cabeçalhos):

    As tags de cabeçalho (heading) são usadas para definir títulos e subtítulos dentro do seu conteúdo. <h1> representa o cabeçalho de nível mais alto (mais importante), enquanto <h6> representa o nível mais baixo.

    Exemplo:

    
    <h1>Título Principal da Página</h1>
    <h2>Subtítulo da Seção</h2>
    <h3>Título de um Subtópico</h3>

  • <p> (Parágrafo):

    A tag <p> é usada para agrupar blocos de texto em parágrafos. Os navegadores geralmente adicionam uma linha em branco antes e depois de cada parágrafo.

    Exemplo:

    
    <p>Este é o primeiro parágrafo do nosso texto. Ele pode conter várias frases e informações sobre o assunto.</p>
    <p>Aqui temos um segundo parágrafo, que aborda um aspecto diferente do tema principal.</p>

  • <br> (Quebra de Linha):

    A tag <br> (break line) insere uma quebra de linha simples no texto. É útil quando você precisa iniciar uma nova linha sem iniciar um novo parágrafo.

    Observação: Use <br> com moderação. Para estruturar o conteúdo, geralmente é mais adequado usar parágrafos (<p>).

    Exemplo:

    
    Este é um texto
    
    com uma quebra de linha forçada.

  • <hr> (Linha Horizontal):

    A tag <hr> (horizontal rule) cria uma linha horizontal que pode ser usada para separar visualmente diferentes seções de conteúdo.

    Exemplo:

    
    <p>Primeira seção do conteúdo.</p>
    <hr>
    <p>Segunda seção do conteúdo.</p>

  • Tags de Formatação Semântica:

    • <strong> (Texto Importante):

      Indica que o texto tem forte importância, seriedade ou urgência. Os navegadores geralmente exibem esse texto em negrito.

      Exemplo: <p>Esta é uma informação <strong>muito importante</strong>.</p>

    • <em> (Ênfase):

      Indica ênfase no texto. Os navegadores geralmente exibem esse texto em itálico.

      Exemplo: <p>Você <em>precisa</em> ver este ponto.</p>

    • <mark> (Texto Marcado):

      Representa texto marcado ou destacado para referência ou anotação.

      Exemplo: <p>A palavra chave é <mark>HTML</mark>.</p>

    • <small> (Texto Menor):

      Representa um comentário lateral pequeno, como informações de direitos autorais ou outros avisos.

      Exemplo: <p><small>© 2023 Meu Site</small></p>

    • <del> (Texto Deletado):

      Representa texto que foi deletado do documento. Os navegadores geralmente o exibem riscado.

      Exemplo: <p>Preço antigo: <del>R$ 29,90</del> Novo preço: R$ 19,90</p>

    • <ins> (Texto Inserido):

      Representa texto que foi inserido no documento. Os navegadores geralmente o exibem sublinhado.

      Exemplo: <p>Agora com <ins>novas funcionalidades</ins>!</p>

  • Tags para Citações e Código:

    • <blockquote> (Citação em Bloco):

      Usada para citar uma seção longa de outro recurso. Os navegadores geralmente a renderizam com recuo.

      Exemplo:

      
    <blockquote cite="https://www.exemplo.com/artigo">
      
      Esta é uma citação longa de um artigo externo.
      
    </blockquote>
    • <q> (Citação Curta):

      Usada para citações curtas que não precisam de quebra de linha.

      Exemplo: <p>Como diz o ditado, <q>água mole em pedra dura, tanto bate até que fura.</q></p>

    • <pre> (Texto Pré-formatado):

      Exibe o texto exatamente como foi escrito no código HTML, preservando espaços em branco e quebras de linha. É útil para exibir código.

      Exemplo:

      
    function minhaFuncao() {
      console.log("Olá!");
    }
    • <code> (Fragmento de Código):

      Usada para marcar pequenos trechos de código dentro de um texto.

      Exemplo: Para imprimir algo no console, use a função <code>console.log()</code>.

Considerações Importantes:

  • Use as tags de cabeçalho (<h1> a <h6>) de forma hierárquica para estruturar o conteúdo da sua página. <h1> deve ser usado para o título principal.
  • Escolha as tags semânticas (como <strong> e <em>) para transmitir o significado do texto, em vez de usar apenas tags de apresentação (como <b> para negrito e <i> para itálico), a menos que a intenção seja puramente estilística.
  • Mantenha o conteúdo textual legível e bem organizado para melhorar a experiência do usuário e a acessibilidade.

Tipos de Input de Texto em HTML

A tag <input> é um dos elementos de formulário mais poderosos e versáteis do HTML. O atributo type dentro da tag <input> define o tipo de controle de entrada a ser exibido. Para campos de texto, existem diversos valores para o atributo type que oferecem funcionalidades específicas.

Principais Tipos de Input de Texto:

  • type="text" (Texto Simples):

    O que faz: Cria um campo de entrada de texto de linha única para entrada de texto genérico, como nomes, endereços curtos, etc.

    Características: Não possui validação específica ou formatação automática.

    Exemplo:

    
    <label for="nome">Nome:<br>
    <input type="text" id="nome" name="nome">

  • type="password" (Senha):

    O que faz: Cria um campo de entrada para senhas. Os caracteres digitados são geralmente mascarados (substituídos por asteriscos ou bolinhas) para segurança.

    Características: O valor digitado não é exibido diretamente na tela.

    Exemplo:

    
    <label for="senha">Senha:<br>
    <input type="password" id="senha" name="senha">

  • type="email" (E-mail):

    O que faz: Cria um campo de entrada otimizado para endereços de e-mail. Os navegadores modernos geralmente performam uma validação básica para garantir que o texto inserido segue o formato de um e-mail válido (ex: alguma coisa@dominio.com).

    Características: Validação básica de formato de e-mail.

    Exemplo:

    
    <label for="email">E-mail:<br>
    <input type="email" id="email" name="email">

  • type="number" (Número):

    O que faz: Cria um campo de entrada para números. Os navegadores geralmente exibem controles para aumentar e diminuir o valor e podem impedir a entrada de caracteres não numéricos.

    Características: Validação para entrada numérica, atributos como min, max e step podem ser usados para restringir os valores.

    Exemplo:

    
    <label for="idade">Idade:<br>
    <input type="number" id="idade" name="idade" min="0" max="150">

  • type="tel" (Telefone):

    O que faz: Cria um campo de entrada para números de telefone. Embora não imponha um formato específico por padrão, navegadores em dispositivos móveis podem otimizar o teclado para entrada de números de telefone.

    Características: Otimização do teclado em dispositivos móveis.

    Exemplo:

    
    <label for="telefone">Telefone:<br>
    <input type="tel" id="telefone" name="telefone">

  • type="url" (URL):

    O que faz: Cria um campo de entrada otimizado para URLs (endereços web). Os navegadores modernos geralmente performam uma validação básica para garantir que o texto inserido segue o formato de um URL válido (ex: https://www.exemplo.com).

    Características: Validação básica de formato de URL.

    Exemplo:

    
    <label for="website">Website:<br>
    <input type="url" id="website" name="website">

  • type="search" (Busca):

    O que faz: Cria um campo de entrada para termos de busca. Alguns navegadores podem exibir um ícone de "limpar" (um "X") dentro do campo.

    Características: Aparência otimizada para busca (ícone de limpar em alguns navegadores).

    Exemplo:

    
    <label for="busca">Buscar:<br>
    <input type="search" id="busca" name="busca">

  • type="range" (Faixa):

    O que faz: Cria um controle deslizante que permite ao usuário selecionar um valor dentro de um intervalo.

    Características: Representação visual de uma faixa de valores, atributos como min, max e step são importantes.

    Exemplo:

    
    <label for="volume">Volume:<br>
    <input type="range" id="volume" name="volume" min="0" max="100" step="10">

  • <textarea> (Área de Texto Multilinha):

    O que faz: Cria um campo de entrada de texto com várias linhas.

    Características: Permite a entrada de texto em múltiplas linhas, atributos como rows e cols definem as dimensões iniciais.

    Exemplo:

    
    <label for="mensagem">Mensagem:<br>
    <textarea id="mensagem" name="mensagem" rows="5" cols="30"></textarea>

Atributos Comuns a Inputs de Texto:

  • id: Identificador único do elemento.
  • name: Nome do elemento (usado para enviar dados do formulário).
  • value: Valor inicial do campo.
  • placeholder: Texto de dica exibido quando o campo está vazio.
  • required: Especifica que o campo deve ser preenchido antes de enviar o formulário.
  • readonly: Especifica que o campo é somente leitura.
  • disabled: Especifica que o campo está desabilitado e não pode ser editado nem enviado.
  • maxlength: Especifica o número máximo de caracteres permitidos no campo.
  • minlength: Especifica o número mínimo de caracteres necessários no campo.
  • size: Especifica a largura visível do campo (em caracteres).

A escolha do tipo de input correto é crucial para fornecer uma boa experiência de usuário e garantir a integridade dos dados do formulário.


Listas ordenadas e não ordenadas

(Voltar)

O HTML oferece duas tags principais para criar listas de itens:

  • Listas Não Ordenadas (Unordered Lists): Usadas para exibir itens em uma lista onde a ordem não é importante. Os itens são geralmente precedidos por marcadores (bullets).
  • Listas Ordenadas (Ordered Lists): Usadas para exibir itens em uma sequência específica. Os itens são numerados ou precedidos por letras.

Listas Não Ordenadas (<ul>):

A tag <ul> (unordered list) define uma lista não ordenada. Cada item da lista dentro de uma lista não ordenada é definido pela tag <li> (list item).

Exemplo de Lista Não Ordenada:


<ul>
  <li>Café</li>
  <li>Chá</li>
  <li>Leite</li>
</ul>

Resultado no navegador:

  • Café
  • Chá
  • Leite

Atributos da tag <ul> (Obsolescente em HTML5, use CSS para estilização):

  • type: Especificava o tipo de marcador a ser usado (disc, circle, square). Este atributo não é mais recomendado em HTML5; a estilização deve ser feita com CSS.

Listas Ordenadas (<ol>):

A tag <ol> (ordered list) define uma lista ordenada. Assim como nas listas não ordenadas, cada item da lista é definido pela tag <li>.

Exemplo de Lista Ordenada:


<ol>
  <li>Primeiro passo</li>
  <li>Segundo passo</li>
  <li>Terceiro passo</li>
</ol>

Resultado no navegador:

  1. Primeiro passo
  2. Segundo passo
  3. Terceiro passo

Atributos da tag <ol>:

  • type: Especifica o tipo de numeração a ser usado:
    • 1 (padrão): Números (1, 2, 3, ...)
    • A: Letras maiúsculas (A, B, C, ...)
    • a: Letras minúsculas (a, b, c, ...)
    • I: Algarismos romanos maiúsculos (I, II, III, ...)
    • i: Algarismos romanos minúsculos (i, ii, iii, ...)
  • start: Especifica o número ou letra inicial da lista. Por exemplo, <ol start="5"> começará a lista com o número 5.
  • reversed: Especifica que a lista deve ser exibida em ordem decrescente (do último item para o primeiro). É um atributo booleano (presente ou ausente).

Exemplo de Lista Ordenada com Atributos:


<ol type="A" start="3" reversed>
  <li>Item C</li>
  <li>Item B</li>
  <li>Item A</li>
</ol>

Resultado no navegador:

  1. Item C
  2. Item B
  3. Item A

Aninhando Listas:

É possível aninhar listas umas dentro das outras para criar estruturas mais complexas.

Exemplo de Listas Aninhadas:


<ul>
  <li>Bebidas</li>
  <ul>
    <li>Quentes</li>
    <ol type="1">
      <li>Café</li>
      <li>Chá</li>
    </ol>
    <li>Frias</li>
    <ul>
      <li>Suco</li>
      <li>Refrigerante</li>
    </ul>
  </ul>
  <li>Comidas</li>
</ul>

Resultado no navegador:

  • Bebidas
    • Quentes
      1. Café
      2. Chá
    • Frias
      • Suco
      • Refrigerante
  • Comidas

Considerações Importantes:

  • Use <ul> quando a ordem dos itens não for significativa.
  • Use <ol> quando a ordem dos itens for importante (passos, sequências, etc.).
  • A tag <li> é essencial para definir cada item dentro de uma lista <ul> ou <ol>.
  • Para estilização avançada das listas (tipos de marcadores, numeração personalizada, etc.), utilize CSS.

Links

(Voltar)

Os links, também conhecidos como hiperlinks, são elementos fundamentais da web. Eles permitem que os usuários naveguem entre diferentes páginas, sites ou recursos online. Em HTML, a tag responsável pela criação de links é a tag de âncora: <a>.

A Tag <a> (Âncora):

A tag <a> cria um hiperlink para outras páginas da web, arquivos, locais dentro da mesma página, endereços de e-mail ou qualquer outro URL.

Atributo Essencial: href

O atributo mais importante da tag <a> é o href (hypertext reference). Ele especifica o destino do link.

Exemplos de Uso do Atributo href:
  • Link para outra página web:
    
<a href="https://www.exemplo.com">Visite Exemplo.com</a>

    Este código cria um link clicável com o texto "Visite Exemplo.com" que direciona o usuário para o site https://www.exemplo.com.

  • Link para um arquivo dentro do mesmo site:
    
<a href="/imagens/logo.png">Veja nosso logo</a>

    Este código cria um link para um arquivo de imagem chamado logo.png localizado na pasta imagens dentro do mesmo site.

  • Link para uma seção específica dentro da mesma página (âncora interna):
    
<h2 id="sobre">Sobre Nós
<p>Informações sobre a nossa empresa...</p>
<a href="#sobre">Ir para a seção Sobre Nós</a>

    Aqui, o atributo href começa com um símbolo # seguido pelo id de outro elemento na mesma página. Ao clicar no link, o navegador rolará até a seção com o id correspondente.

  • Link para um endereço de e-mail:
    
<a href="mailto:contato@exemplo.com">Entre em contato por e-mail</a>

    Ao clicar neste link, o navegador tentará abrir o programa de e-mail padrão do usuário com o endereço contato@exemplo.com no campo "Para".

  • Link para fazer uma chamada telefônica (em dispositivos móveis):
    
<a href="tel:+5534999999999">Ligue para nós</a>

    Em dispositivos móveis, ao clicar neste link, o sistema operacional pode oferecer a opção de fazer uma chamada para o número de telefone especificado.

Atributos Adicionais Úteis:

  • target

    Especifica onde abrir o link.

    • _blank: Abre o link em uma nova janela ou aba do navegador.
    • _self (padrão): Abre o link na mesma janela ou aba.
    • _parent: Abre o link no frame pai.
    • _top: Abre o link na janela inteira (removendo quaisquer frames).

    Exemplo: <a href="https://www.exemplo.com" target="_blank">Visite em nova aba</a>

  • title

    Define um texto que é exibido como uma dica de ferramenta (tooltip) quando o mouse passa sobre o link.

    Exemplo: <a href="#" title="Link para a página inicial">Home</a>

  • rel

    Especifica a relação entre o documento atual e o documento vinculado. É útil para SEO e segurança.

    • noopener: Impede que a nova página tenha acesso ao objeto window.opener da página original, melhorando a segurança.
    • noreferrer: Impede que a nova página receba informações de referência (de onde o usuário veio). Também implica em noopener.
    • nofollow: Sinaliza para os mecanismos de busca que o link não deve influenciar o ranking da página de destino (útil para conteúdo gerado por usuários).

    Exemplo: <a href="https://www.exemplo.com" target="_blank" rel="noopener noreferrer">Link seguro</a>

  • Conteúdo do Link

    O texto ou outros elementos (como imagens) que ficam entre a tag de abertura <a> e a tag de fechamento </a> se tornam o conteúdo clicável do link.

    Exemplo com imagem como link: <a href="https://www.exemplo.com"><img src="logo.png" alt="Logo do Exemplo"></a>

Melhores Práticas para Links:

  • Use texto descritivo para os links, para que os usuários saibam para onde serão direcionados. Evite frases genéricas como "Clique aqui".
  • Use o atributo title para fornecer informações adicionais sobre o link.
  • Considere usar target="_blank" com cuidado, pois pode confundir alguns usuários. Geralmente, é mais intuitivo abrir links externos em novas abas e links internos na mesma aba.
  • Use os atributos rel="noopener noreferrer" para links que abrem em novas abas por questões de segurança e privacidade.
  • Verifique regularmente se seus links estão funcionando corretamente.

    Um site dedicado ao HTML, oferecendo tutoriais, guias e referências de forma clara e organizada. Disponível em inglês.

  • DevMedia (Brasil)

    Portal brasileiro com diversos artigos, tutoriais e cursos sobre desenvolvimento web, incluindo HTML. Conteúdo em português.
